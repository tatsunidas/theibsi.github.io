IBSI Chapter 1 − Radiomic Computations

Researchers can now use the reference values and standardised radiomic computation workflow established by the IBSI 1 to calibrate their own software, by following these instructions:

  1. Read the documentation related to IBSI 1 in the Documentation tab of this website.
  2. Follow the instructions in the IBSI 1 reference manual to perform the tests of Phase 1 and Phase 2 on the datasets available in the Datasets tab of this website.
  3. Compare and validate your results using the IBSI 1 reference values found below.
